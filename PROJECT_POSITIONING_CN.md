# AlphaLens CN 项目定位

## 一句话
一个面向中文用户的开源 A 股复盘与分析工具，帮助交易者、研究者和维护者用结构化方式完成每日市场回顾。

## 核心价值
- 把零散的复盘流程产品化、脚本化、可追溯化
- 降低 A 股日度复盘的人肉整理成本
- 为中文生态提供可维护、可扩展的开源分析基础设施

## MVP 功能
- 每日市场概览
- 涨停/炸板/跌停情绪指标
- 热门概念聚合
- 次日观察池生成
- Markdown 日报导出

## 后续可扩展方向
- AKShare / 东方财富数据接入
- 龙虎榜分析
- 机构连续净买入线索
- 题材梯队识别
- 自动化日报调度
- Web 面板 / API 服务

## 为什么适合申请 Codex for OSS
- 属于真实的持续维护型项目
- 有代码审查、数据适配、文档维护、测试补全等长期工作
- 可以自然使用 Codex 做 review、重构、自动化和安全检查
